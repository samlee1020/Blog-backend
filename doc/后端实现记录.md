# 后端实现记录

## 已实现范围

- Spring Boot 3 + MyBatis-Plus + MySQL + Redis + Flyway 项目骨架。
- 统一响应、分页响应、业务异常、参数校验和全局异常处理。
- 游客注册、管理员/游客登录、Redis token 登录态、登录失败限流、退出登录、当前用户查询。
- Spring Security token 鉴权，`/api/admin/**` 仅管理员访问，游客/管理员可发表评论。
- 文章、项目、分类、标签、评论、游客用户、站点配置、系统配置、图片上传接口。
- 文章和评论使用软删除；文章创建/更新会同步维护标签关系。
- 图片上传写入本地目录和 `media_asset`，数据库失败时删除已保存文件。
- Flyway 初始化表结构、默认封面、默认信息页、基础系统配置。
- Dockerfile 与 `docker-compose.yml`。

## 重要实现约定

- 密码只保存 BCrypt 哈希，任何响应不返回明文密码或 `passwordHash`。
- 管理员只能重置游客密码，不能查看原密码。
- 管理员账号在启动时按 `APP_ADMIN_USERNAME` / `APP_ADMIN_PASSWORD` 自动创建。
- `contentHtml` 第一版不做后端 Markdown 渲染，返回 `null`。
- 评论内容保存前进行基础 HTML escape。
- Redis 缓存包含文章列表、文章详情、封面、信息页；配置或文章变更后删除相关缓存。

## 2026-05-05 项目管理模块

- 根据 `doc/项目类添加要求.md` 新增项目管理后端能力。
- 新增 `project` 表迁移：`V2__add_project.sql`。
- 新增 `ProjectStatus`：`DRAFT`、`PUBLISHED`、`HIDDEN`。
- 新增公开接口：`GET /api/projects`、`GET /api/projects/{slug}`。
- 新增后台接口：`GET /api/admin/projects`、`POST /api/admin/projects`、`GET /api/admin/projects/{id}`、`PUT /api/admin/projects/{id}`、`DELETE /api/admin/projects/{id}`。
- 新增 `MediaUsageType.PROJECT`，项目图片上传可复用 `POST /api/admin/media/images`。
- 新增 API 文档：`doc/项目管理新增API文档.md`。

## 2026-05-05 项目长说明字段

- 给 `project` 表新增 `content_markdown` 字段，用于保存可选 Markdown 长正文。
- 新增迁移：`V3__add_project_content_markdown.sql`。
- `ProjectRequest` / `ProjectView` 新增 `contentMarkdown` 字段。
- 项目创建、更新、公开列表、公开详情、后台列表、后台详情均会读写/返回 `contentMarkdown`。

## 2026-05-05 项目详情入口字段

- `ProjectView` 新增只读字段 `detailUrl`，固定返回 `/projects/{slug}`。
- 前端项目卡片整体点击应进入 `detailUrl` 对应的项目详情页。
- `projectUrl` 保持作为外部项目链接，建议仅用于“访问项目 / GitHub / Demo”等显式按钮，避免覆盖详情入口。
