- 当前本机未安装 Docker Compose 插件，因此未执行 `docker compose up --build` 全链路命令；该项已在第二轮解决。
- Flyway 在 MySQL 8.4 上提示版本支持和部分 SQL 写法弃用警告，不影响本轮启动和迁移；后续可将 compose 镜像固定到 MySQL 8.1 或调整 SQL 写法。
- 文章列表缓存清理仍使用 Redis `keys`，数据量大时建议替换为版本号或 `scan`。
